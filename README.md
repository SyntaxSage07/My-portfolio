##  About The Project

This portfolio was built from scratch without any frameworks or libraries.
The goal was to demonstrate strong fundamentals in:

* Semantic HTML5
* Modern CSS (Flexbox & Grid)
* Responsive Design
* Vanilla JavaScript Interactivity
* Clean Project Architecture

The website is fully responsive and optimized for desktop, tablet, and mobile devices.


##  Built With

* HTML5
* CSS3
* JavaScript (ES6+)
* Lucide Icons

No frameworks. No Bootstrap. No React. Pure frontend engineering.
